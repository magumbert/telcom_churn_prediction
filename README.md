# TELCOM churn prediction
This script leverages machine learning techniques (PyCaret, Sklearn, QDA, XGBoost...) to predict and analyse customer churn behavior. The dataset is provided by Kaggle. Info about the dataset can be found here: https://community.ibm.com/community/user/businessanalytics/blogs/steven-macko/2019/07/11/telco-customer-churn-1113.
